# Veilwright MCP

Pre-built, ready-to-run distribution of Veilwright MCP — no TypeScript source, no build step needed.

**Setup: on macOS, right-click `setup-mac.command` → Open (not a plain double-click — see docs/self-hosting.md for why); on Windows, double-click `setup-windows.bat`; on Linux, run `./setup-linux.sh`.** Installs dependencies, generates the vault key, and wires up Claude Desktop's config automatically — no terminal commands to type. Full guide (including manual steps if you'd rather): [`docs/self-hosting.md`](docs/self-hosting.md).

Full source (private) is developed at github.com/pixagentur/veilwright-mcp — this repo only ever receives compiled releases, published automatically from that repo's tags.
