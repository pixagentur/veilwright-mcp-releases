# Veilwright MCP

Pre-built, ready-to-run distribution of Veilwright MCP — no TypeScript source, no build step needed.

**Setup: double-click `setup-mac.command` (macOS), `setup-windows.bat` (Windows), or run `./setup-linux.sh` (Linux).** Installs dependencies, generates the vault key, and wires up Claude Desktop's config automatically — no terminal commands to type. **On macOS it will be blocked the first time ("Apple could not verify...") — this is expected, see docs/self-hosting.md step 3 for the exact (still no-Terminal) fix via System Settings.** Full guide (including manual steps if you'd rather): [`docs/self-hosting.md`](docs/self-hosting.md).

Full source (private) is developed at github.com/pixagentur/veilwright-mcp — this repo only ever receives compiled releases, published automatically from that repo's tags.
